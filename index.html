Arrays • Stacks • Queues

Arrays

An array is a fixed-size, ordered collection of elements stored in contiguous memory locations. Each element can be accessed by its index, starting from 0.

Stack

A stack follows the LIFO principle (Last In, First Out). The last item pushed onto the stack is the first one to be popped. Think of it like a stack of books. As you add books, they go on top, and when you remove a book, you take the top one off.

Queue

A queue follows the FIFO principle (First In, First Out). The first item added is the first one removed. Think of it like a line of people waiting. the first person in line is the first to be served.
