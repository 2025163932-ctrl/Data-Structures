Arrays • Stacks • Queues

Arrays

An array is a fixed-size, ordered collection of elements stored in contiguous memory locations. Each element can be accessed by its index, starting from 0.

Stack

A stack follows the LIFO principle (Last In, First Out). The last item pushed onto the stack is the first one to be popped.

Queue

A queue follows the FIFO principle (First In, First Out). The first item added is the first one removed.
